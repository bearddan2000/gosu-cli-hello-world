# gosu-cli-hello-world

## Description
A POC for gosu programming language.

## Tech stack
- groovy
- java

## Docker stack
- openjdk:11
- rzuckerm/gosu

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
